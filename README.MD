此应用由孟诗远个人独立开发，可用于个人用途，未经授权严禁用于商业用途

# 应用功能
* 可将m3u8格式的Apple HLS加密视频流缓存文件转换为mp4格式  
    能大幅减少文件数量，降低文件系统开销  
    能略微减少文件体积
* 支持UC浏览器缓存文件夹、QQ浏览器缓存文件夹自动搜索
* 支持文件多选和列表文件批量转换
* 支持查看m3u8文件的详细信息
